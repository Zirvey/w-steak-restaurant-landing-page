UI / UX

Klíčová vylepšení rozhraní

Hero sekce

Fullscreen fotografie, jasný nadpis a dvě CTA tlačítka: „Book a Table“ a „Explore Menu“. Host okamžitě chápe hodnotu a další krok.

Glass navigace

Plovoucí menu se skleněným efektem, adaptace při scrollu, sticky tlačítko Reservations — konverze dostupná na každé obrazovce.

Bento Experience

Čtyři karty: Meat · Wine · Whiskey · Ambience — vizuálně představují jedinečnost restaurace bez dlouhého textu.

Menu na homepage

Top položky s fotografiemi a cenami v CZK. Snižuje bariéru výběru a motivuje k přechodu na celé menu.

Animace

Plynulé scroll-reveal efekty, statistické počítadla, hover efekty — živý prémiový zážitek bez přetížení.

Design systém

Playfair Display SC + Karla, paleta #231F20 / #DB821F — v souladu s firemním stylem W Restaurant.

Mobile First

Mobilní verze — priorita

  • ✦  Responzivní layout — od 375px do 1440px+ bez horizontálního scrollu
  • ✦  Burger menu — fullscreen navigace s animací, velké dotykové zóny
  • ✦  Click-to-call — telefon +420 730 779 999 jedním klepnutím
  • ✦  Sticky CTA — tlačítko „Reservations“ vždy dole na obrazovce
  • ✦  Optimalizované obrázky — rychlé načítání na 4G
  • ✦  Galerie tahem prstu — horizontální scroll gestem

Většina turistů hledá restauraci u Staroměstského náměstí z mobilu. Mobilní UX přímo ovlivňuje počet rezervací.

Technologie

Stack a architektura

Lehký statický stack bez těžkých frameworků — maximální rychlost, snadná údržba a nízké náklady na hosting.

HTML5 CSS3 Vanilla JavaScript Tailwind CSS (CDN) Google Fonts Intersection Observer API

Proč ne Tilda / builder?

  • Plná kontrola nad UX a animacemi
  • Žádné měsíční předplatné platformy
  • SEO optimalizace na úrovni kódu
  • Integrace s libovolným CRM / booking systémem
  • Škálování: vícejazyčnost, A/B testy

Výkon

  • Bez React/Vue — minimální JS bundle
  • Statika → CDN / GitHub Pages / Cloudflare
  • Optimalizované obrázky, rychlá struktura
  • Core Web Vitals: rychlý LCP, žádný layout shift

Přístupnost (a11y)

  • Sémantické HTML a alt texty
  • Kontrast textu WCAG AA
  • Keyboard navigace + focus stavy
  • prefers-reduced-motion pro citlivé uživatele

Obchodní hodnota

Co získá restaurace

Konverze

CTA „Reservations“ v hero, navigaci, sticky baru a patičce — host vždy vidí cestu k rezervaci.

Značka

Vizuál úrovně fine dining: tmavé tóny, zlato, typografie — odpovídá cenovému segmentu.

Důvěra

Profesionální prezentace menu, whiskey baru a interiéru zvyšuje vnímanou hodnotu.

SEO

Čisté HTML, meta tagy, strukturovaný obsah — lepší indexace v Google.

Roadmapa implementace

  • Fáze 1 — Landing page (hotovo jako koncept)
  • Fáze 2 — Stránky Menu, Gallery, Events
  • Fáze 3 — Integrace rezervací + CZ/EN
  • Fáze 4 — Analytika, SEO, A/B testy CTA

Hotový prototyp

Interaktivní koncept je již k dispozici k prohlížení. Lze otestovat s týmem, získat zpětnou vazbu a rozhodnout o nasazení.
